# AUREN DI BRAID
*Sentinella di Aethon, formato alla scuola di Sohl*

| Campo | Valore |
|---|---|
| **Fazione** | Nord + Centro |
| **Tipo** | Monaco · Umano · Leader |
| **Costo mana** | 1 Nord + 1 Centro + 1 (Nord o Centro) |
| **ATK / DEF** | 1 / 5 |
| **Keyword** | Scudo |
| **Passiva** | "Le tue unità con Scudo non possono essere bersaglio di effetti avversari." |
| **Hero Power** | Ogni 3 turni, 1 Nord + 1 Centro — *"Applica Scudo a una tua unità."* |
| **Evoluzione** | Quando controlli 4 o più unità con Scudo → **AUREN, DIFENSORE ARCANO** ATK 3 / DEF 8 — ogni volta che una tua creatura con Scudo muore, guadagni 2 HP |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 3 turni → costo originale |
