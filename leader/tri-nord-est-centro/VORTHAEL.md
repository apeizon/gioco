# VORTHAEL, SENTENZA DELL'OBLIO
*Giudice di Aethon corrotto dall'oscurità di Endal — il gelo del Nord ne custodisce la forma, il fuoco divino ne alimenta la condanna*

| Campo | Valore |
|---|---|
| **Fazione** | Nord + Est + Centro |
| **Tipo** | Divinità · Leader |
| **Costo mana** | 1 Nord + 2 Est + 1 Centro |
| **ATK / DEF** | 2 / 5 |
| **Keyword** | Baluardo |
| **Passiva** | "Ogni volta che un'unità avversaria congelata viene distrutta, guadagni 3 HP." |
| **Hero Power** | Ogni 2 turni, 3 mana a scelta — *"Congela un'unità avversaria per 1 turno e applica Scudo a una tua unità."* |
| **Evoluzione** | Paga 10 HP → **VORTHAEL, L'ULTIMA SENTENZA** ATK 4 / DEF 7 — la Passiva si attiva su qualsiasi unità che muore, non solo congelate (sempre 3 HP) |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 3 turni → costo originale |
