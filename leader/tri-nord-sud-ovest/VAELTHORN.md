# VAELTHORN, PATTO DELLE STAGIONI
*Elementale di Faelorn che ha stretto un antico accordo con le forze elementali di Vorthaan e Caleth*

| Campo | Valore |
|---|---|
| **Fazione** | Nord + Sud + Ovest |
| **Tipo** | Elementale · Leader |
| **Costo mana** | 2 Nord + 1 Sud + 2 Ovest |
| **ATK / DEF** | 3 / 4 |
| **Keyword** | Vedetta |
| **Passiva** | "Le tue unità guadagnano +0/+1 per ogni tipo di fazione diversa nel loro costo mana." |
| **Hero Power** | Ogni 3 turni, 3 mana a scelta — *"Applica +0/+1 permanente a tutte le tue unità in campo."* |
| **Evoluzione** | Quando controlli almeno 1 unità Nord, 1 Sud e 1 Ovest simultaneamente → **VAELTHORN, EQUILIBRIO SPEZZATO** ATK 5 / DEF 6 — la Passiva applica anche +1/+0 |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 3 turni → costo originale |
