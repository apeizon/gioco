# SERATH DEL FUOCO CORROTTO
*Demone del deserto corrotto dal sangue di Endal*

| Campo | Valore |
|---|---|
| **Fazione** | Sud + Est |
| **Tipo** | Demone · Assassino · Leader |
| **Costo mana** | 1 Sud + 1 Est + 1 (Sud o Est) |
| **ATK / DEF** | 4 / 3 |
| **Keyword** | Velocità |
| **Passiva** | "Ogni volta che un'unità avversaria muore, applica 1 segnalino Sanguinamento al suo controllore." |
| **Hero Power** | Ogni 2 turni, 1 Sud + 1 Est — *"Sacrifica una tua unità: infligge 2 danni a un target a scelta."* |
| **Evoluzione** | Rimuovi 7 segnalini Sanguinamento dal campo di battaglia (senza applicarne gli effetti) → **SERATH, BRACIERE MALEDETTO** ATK 6 / DEF 4 — la Passiva applica 2 segnalini invece di 1 |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 2 turni → costo originale |
