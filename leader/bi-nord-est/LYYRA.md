# LYYRA, IL RISORTO
*Drakar del Nord corrotta dall'oscurità di Endal*

| Campo | Valore |
|---|---|
| **Fazione** | Nord + Est |
| **Tipo** | Drago · Zombie · Leader |
| **Costo mana** | 1 Nord + 2 Est + 2 (Nord o Est) |
| **ATK / DEF** | 3 / 5 |
| **Keyword** | Volo |
| **Passiva** | "Ogni volta che Lyyra attacca, crea un token **Uovo 0/0** nel tuo campo. Le Uova sono creature senza abilità, bersagliabili come unità normali." |
| **Hero Power** | Ogni 4 turni, 1 Nord + 1 Est — *"Applica un segnalino +1/+1 a tutti i Draghi che controlli."* |
| **Evoluzione** | Quando controlli 4 o più Uova, puoi pagare 5 HP in qualsiasi momento → le Uova si trasformano in token **Spettro di Cucciolo di Drago 2/2** — **LYYRA, EVOCATRICE DI ANIME** ATK 6 / DEF 5 |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 4 turni → costo originale |
