# KAELTHAR, IL SIGILLO SPEZZATO
*Angelo del Conclave di Mira divorato dall'oscurità di Endal — il fuoco che portava come dono ora brucia chi lo segue*

| Campo | Valore |
|---|---|
| **Fazione** | Sud + Est + Centro |
| **Tipo** | Angelo · Demone · Leader |
| **Costo mana** | 1 Sud + 2 Est + 1 Centro |
| **ATK / DEF** | 3 / 4 |
| **Keyword** | Volo |
| **Passiva** | "Ogni volta che giochi una carta Sud, infligge 1 danno a tutte le unità avversarie." |
| **Hero Power** | Ogni 3 turni, 2 Sud + 1 Centro — *"Infligge 3 danni a un'unità avversaria e guadagni 2 HP."* |
| **Evoluzione** | Quando hai inflitto 10 danni totali tramite la Passiva → **KAELTHAR, FIAMMA DELL'ABISSO** ATK 5 / DEF 6 — la Passiva infligge 2 danni invece di 1 |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 3 turni → costo originale |
