# MORVRETH, IL SILENZIO
*Lich-divinità che precede il fuoco — il gelo, la morte, la foresta e l'ordine esistevano prima che Caleth bruciasse per la prima volta*

| Campo | Valore |
|---|---|
| **Fazione** | Nord + Est + Ovest + Centro |
| **Tipo** | Lich · Divinità · Leader |
| **Costo mana** | 2 Nord + 2 Est + 1 Ovest + 1 Centro |
| **ATK / DEF** | 2 / 7 |
| **Keyword** | Protezione — *"Non può essere bersaglio di effetti avversari. Non protegge da effetti non mirati (es. distruggi tutte le creature)."* |
| **Passiva** | "All'inizio del tuo turno, guarda le prime 3 carte del tuo mazzo: puoi mettere 1 in mano scartandone 1 dalla tua mano, e rimettere le altre in cima in qualsiasi ordine." |
| **Hero Power** | Ogni 4 turni, 1 Nord + 1 Ovest — *"Cerca nel tuo mazzo una carta creatura con DEF 3 o più, mettila in cima al mazzo e mischia il mazzo."* |
| **Evoluzione** | Quando hai pescato o cercato 8 carte tramite Passiva e Hero Power → **MORVRETH, RESPIRO GELIDO** ATK 4 / DEF 9 — la Passiva guarda le prime 5 carte invece di 3 |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 4 turni → costo originale |
