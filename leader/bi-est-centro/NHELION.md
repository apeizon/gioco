# NHELION, L'EQUILIBRIO CORROTTO
*Divinità di Aethon caduta nell'oscurità di Endal — ciò che un tempo proteggeva ora consuma*

| Campo | Valore |
|---|---|
| **Fazione** | Est + Centro |
| **Tipo** | Divinità · Costrutto · Leader |
| **Costo mana** | 3 Est + 3 Centro |
| **ATK / DEF** | 3 / 4 |
| **Keyword** | Assorbimento |
| **Passiva** | "Ogni volta che un'unità sul campo di battaglia muore, guadagni 1 HP." |
| **Hero Power** | Ogni 3 turni, 1 Est + 1 Centro — *"Applica 2 segnalini Sanguinamento a un'unità avversaria e guadagna 1 HP per ogni segnalino Sanguinamento già presente su di essa."* |
| **Evoluzione** | Se Nhelion ha già attaccato in questo turno, al turno successivo puoi scegliere di non stapparlo per trasformarlo → **NHELION, DIO DEL CREPUSCOLO** ATK 5 / DEF 6 |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 3 turni → costo originale |
