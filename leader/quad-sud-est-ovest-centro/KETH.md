# KETH, IL VASO SENZIENTE
*Costrutto creato da Aethon per contenere le quattro energie di Solvarath — quando si spezza, le libera tutte insieme*

| Campo | Valore |
|---|---|
| **Fazione** | Sud + Est + Ovest + Centro |
| **Tipo** | Costrutto · Leader |
| **Costo mana** | 1 Sud + 1 Est + 1 Ovest + 2 Centro |
| **ATK / DEF** | 3 / 6 |
| **Keyword** | Esplosione 4 — *"Quando viene distrutto, infligge 4 danni a tutte le unità avversarie in campo."* |
| **Passiva** | "Ogni volta che questa unità subisce danno, accumula 1 segnalino Carica. Per ogni 3 segnalini Carica, guadagna +1/+1 permanente." |
| **Hero Power** | Ogni 3 turni, 2 Centro — *"Trasferisci tutti i segnalini Carica di Keth a una tua unità a scelta come +1/+1 permanenti."* |
| **Evoluzione** | Quando ha 9 o più segnalini Carica → **KETH, IL VASO SPEZZATO** ATK 5 / DEF 8 — Esplosione diventa 6 danni invece di 4 |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 4 turni → costo originale |
