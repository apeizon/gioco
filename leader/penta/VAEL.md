# VAEL, GUARDIANO DEI SEI
*La divinità incaricata di proteggere i Sette Sigilli — ne ha perso uno, non perderà gli altri*

| Campo | Valore |
|---|---|
| **Fazione** | Tutte e 5 |
| **Tipo** | Divinità · Costrutto · Leader |
| **Costo mana** | 2 Nord + 1 Sud + 1 Est + 2 Ovest + 2 Centro |
| **ATK / DEF** | 4 / 7 |
| **Keyword** | Baluardo — *"Finché è in campo, tutte le altre tue unità ricevono +0/+1 DEF."* |
| **Passiva** | "Ogni volta che un'unità avversaria attacca, puoi scegliere di far intercettare l'attacco a Vael, indipendentemente dal bersaglio dichiarato." |
| **Hero Power** | Ogni 3 turni, 2 Centro + 1 a scelta — *"Cerca nel tuo mazzo una carta con Scudo o Baluardo e mettila in mano."* |
| **Evoluzione** | Quando Vael ha intercettato 5 attacchi → **VAEL, L'ULTIMO SCUDO** ATK 6 / DEF 9 — ogni attacco intercettato infligge 2 danni all'attaccante |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 5 turni → costo originale |
