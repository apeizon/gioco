# AETHVAEL, CUORE DI SOLVARATH
*Guerriero-divinità che incarna le quattro forze viventi di Solvarath — l'unico in grado di tenerle unite*

| Campo | Valore |
|---|---|
| **Fazione** | Nord + Sud + Ovest + Centro |
| **Tipo** | Guerriero · Divinità · Leader |
| **Costo mana** | 2 Nord + 1 Sud + 2 Ovest + 1 Centro |
| **ATK / DEF** | 3 / 6 |
| **Keyword** | Baluardo — *"Finché questa unità è in campo, tutte le altre tue unità ricevono +0/+1 DEF."* |
| **Passiva** | "Le tue unità guadagnano Scudo quando entrano in campo se controlli già almeno 4 unità." |
| **Hero Power** | Ogni 3 turni, 1 Nord + 1 Ovest + 1 Centro — *"Tutte le tue unità guadagnano +0/+1 permanente."* |
| **Evoluzione** | Quando controlli 5 o più unità con Scudo → **AETHVAEL, ULTIMO BALUARDO** ATK 5 / DEF 8 — la Passiva si attiva con 3 unità invece di 4 |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 4 turni → costo originale |
