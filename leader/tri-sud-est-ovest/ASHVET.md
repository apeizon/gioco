# ASHVET, DRUIDO DELLE CENERI
*Druido-bestia nato dalle frontiere bruciate tra Caleth e Faelorn — si nutre di fuoco e sangue*

| Campo | Valore |
|---|---|
| **Fazione** | Sud + Est + Ovest |
| **Tipo** | Druido · Bestia · Leader |
| **Costo mana** | 2 Sud + 1 Est + 1 Ovest |
| **ATK / DEF** | 4 / 3 |
| **Keyword** | Velocità |
| **Passiva** | "Ogni volta che questa unità infligge danno diretto al giocatore avversario, infligge 1 danno a un target a scelta." |
| **Hero Power** | Ogni 3 turni, 2 Sud + 1 a scelta — *"Una tua creatura a scelta guadagna Velocità e +1/+1 fino alla fine del turno."* |
| **Evoluzione** | Quando hai inflitto 10 danni diretti al giocatore avversario → **ASHVET, LA CACCIA ETERNA** ATK 6 / DEF 4 — la Passiva infligge 2 danni invece di 1 |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 3 turni → costo originale |
