*Guerriero di Vorath's Gate, forgiato tra ghiaccio e fiamme*

| Campo | Valore |
|---|---|
| **Fazione** | Nord + Sud |
| **Tipo** | Guerriero · Elementale · Leader |
| **Costo mana** | 1 Nord + 1 Sud + 1 (Nord o Sud a scelta) |
| **ATK / DEF** | 3 / 3 |
| **Keyword** | Velocità |
