# SOLANDRA, DEL VELO DORATO
*Elfa di Faelorn consacrata ad Aethon — porta il fuoco di Caleth come nutrimento, non come distruzione*

| Campo | Valore |
|---|---|
| **Fazione** | Sud + Ovest + Centro |
| **Tipo** | Elfo · Divinità · Leader |
| **Costo mana** | 2 Ovest + 1 Sud + 1 Centro |
| **ATK / DEF** | 3 / 4 |
| **Keyword** | Prescienza |
| **Passiva** | "Ogni volta che giochi una carta Ovest, una tua unità Sud guadagna +1/+1 permanente." |
| **Hero Power** | Ogni 3 turni, 1 Sud + 1 Centro — *"Crea un token Fiamma Sacra 2/1 con Velocità nel tuo campo."* |
| **Evoluzione** | Quando controlli 3 o più token in campo → **SOLANDRA, DELLA SCINTILLA** ATK 5 / DEF 6 — la Passiva si applica anche al contrario: ogni carta Sud giocata applica +1/+1 a un'unità Ovest |
| **Rientro A** | Costo originale +2 mana per ogni morte |
| **Rientro B** | Aspetta 3 turni → costo originale |
