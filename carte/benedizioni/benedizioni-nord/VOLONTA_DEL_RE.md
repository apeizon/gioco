# Volontà del Re

| Campo | Valore |
|---|---|
| **Tipo** | Benedizione |
| **Fazione** | Nord |
| **Rarità** | Eterno |
| **Costo** | 2 Nord + 5 |
| **Effetto** | Tutte le tue creature guadagnano **Scudo** (annulla la prima fonte avversaria che la bersaglia o la colpisce, senza effetto residuo; consumabile a uso singolo). All'inizio del tuo Mantenimento, applica **Gelo 1** (l'unità non può attaccare o attivare abilità per 1 turno) a tutte le unità avversarie. |
