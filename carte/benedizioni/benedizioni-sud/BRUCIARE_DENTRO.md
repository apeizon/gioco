# Bruciare Dentro

| Campo | Valore |
|---|---|
| **Tipo** | Benedizione |
| **Fazione** | Sud |
| **Rarità** | Eterno |
| **Costo** | 2 Sud + 4 |
| **Effetto** | All'inizio del tuo Mantenimento, infliggi 2 danni a ogni avversario. Tutte le tue creature guadagnano **Frenesia** (questa unità può attaccare due volte per turno; la seconda volta non può essere bloccata) fino alla fine del turno. |
