# Lanterna del Pellegrino

| Campo | Valore |
|---|---|
| **Tipo** | Satellite |
| **Fazione** | Nomade |
| **Rarità** | Non Comune |
| **Costo** | 3 mana qualsiasi |
| **Effetto** | All'inizio del tuo Mantenimento, guarda la prima carta del tuo mazzo. Scegli una di queste opzioni: lasciala in cima al mazzo, mettila in fondo al mazzo, oppure mettila nel tuo cimitero. |
