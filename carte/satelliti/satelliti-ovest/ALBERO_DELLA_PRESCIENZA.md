# Albero della Prescienza

| Campo | Valore |
|---|---|
| **Tipo** | Satellite |
| **Fazione** | Ovest |
| **Rarità** | Eterno |
| **Costo** | 1 Ovest + 3 |
| **Effetto** | Tutte le tue creature guadagnano +1 DEF e **Prescienza** (quando questa unità entra in campo, guarda le prime 3 carte del tuo mazzo e puoi riordinarle come vuoi). All'inizio del tuo Mantenimento, pesca 1 carta. |
