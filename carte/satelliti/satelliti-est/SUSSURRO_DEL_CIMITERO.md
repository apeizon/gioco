# Sussurro del Cimitero

| Campo | Valore |
|---|---|
| **Tipo** | Satellite |
| **Fazione** | Est |
| **Rarità** | Non Comune |
| **Costo** | 2 Est + 2 |
| **Effetto** | Ogni volta che una tua creatura entra nel cimitero, l'avversario subisce mill X (scarta le prime X carte del mazzo al cimitero), dove X è pari al costo totale in mana della creatura entrata nel cimitero. |
