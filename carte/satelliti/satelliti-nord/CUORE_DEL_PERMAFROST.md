# Cuore del Permafrost

| Campo | Valore |
|---|---|
| **Tipo** | Satellite |
| **Fazione** | Nord |
| **Rarità** | Eterno |
| **Costo** | 3 Nord + 3 |
| **Effetto** | Tutte le tue creature guadagnano +1 DEF e **Vedetta** (questa unità non viene tappata quando attacca). All'inizio del tuo Mantenimento, applica **Gelo 1** (l'unità non può attaccare o attivare abilità per 1 turno) a un'unità avversaria bersaglio. |
