# Savana Addormentata

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Caleth · Faelorn |
| **Effetto** | Entra inattivo — non genera mana finché la condizione non è soddisfatta. Si attiva quando giochi 3 o più carte in un singolo turno. Una volta attivo, genera 1 mana Sud o 1 mana Ovest a scelta per turno. |
| **Utilizzo** | Illimitato — 1 volta per turno (solo se attivo) |
