# Caldera Sopita

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Sud |
| **Effetto** | Entra inattivo — non genera mana finché la condizione non è soddisfatta. Si attiva la prima volta che infliggi 3 o più danni in un singolo turno. Una volta attivo, genera 1 mana Sud per turno. |
| **Utilizzo** | Illimitato — 1 volta per turno (solo se attivo) |
