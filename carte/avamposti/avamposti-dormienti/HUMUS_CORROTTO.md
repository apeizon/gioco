# Humus Corrotto

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Endal · Faelorn |
| **Effetto** | Entra inattivo — non genera mana finché la condizione non è soddisfatta. Si attiva quando muore una creatura (tua o avversaria) per la prima volta in partita. Una volta attivo, genera 1 mana Est o 1 mana Ovest a scelta per turno. |
| **Utilizzo** | Illimitato — 1 volta per turno (solo se attivo) |
