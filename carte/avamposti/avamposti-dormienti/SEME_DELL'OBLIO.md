# Seme Dell'Oblio

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Ovest |
| **Effetto** | Entra inattivo — non genera mana finché la condizione non è soddisfatta. Si attiva quando controlli 3 o più creature in campo. Una volta attivo, genera 1 mana Ovest per turno. |
| **Utilizzo** | Illimitato — 1 volta per turno (solo se attivo) |
