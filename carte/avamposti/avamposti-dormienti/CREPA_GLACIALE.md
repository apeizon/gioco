# Crepa Glaciale

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Nord |
| **Effetto** | Entra inattivo — non genera mana finché la condizione non è soddisfatta. Si attiva quando hai 4 o più carte in mano. Una volta attivo, genera 1 mana Nord per turno. |
| **Utilizzo** | Illimitato — 1 volta per turno (solo se attivo) |
