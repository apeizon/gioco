# Tundra Sopita

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Nord · Faelorn |
| **Effetto** | Entra inattivo — non genera mana finché la condizione non è soddisfatta. Si attiva quando controlli 3 o più creature e 3 o più Avamposti. Una volta attivo, genera 1 mana Nord o 1 mana Ovest a scelta per turno. |
| **Utilizzo** | Illimitato — 1 volta per turno (solo se attivo) |
