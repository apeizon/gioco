# Cenere Di Mordeth

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Caleth · Endal |
| **Effetto** | Entra inattivo — non genera mana finché la condizione non è soddisfatta. Si attiva la prima volta che uccidi una creatura avversaria. Una volta attivo, genera 1 mana Sud o 1 mana Est a scelta per turno. |
| **Utilizzo** | Illimitato — 1 volta per turno (solo se attivo) |
