# Cripta Velata

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Endal · Mid Vael |
| **Effetto** | Entra inattivo — non genera mana finché la condizione non è soddisfatta. Si attiva quando milli 2 o più carte in un singolo turno. Una volta attivo, genera 1 mana Est o 1 mana Centro a scelta per turno. |
| **Utilizzo** | Illimitato — 1 volta per turno (solo se attivo) |
