# Tomba Silente

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Est |
| **Effetto** | Entra inattivo — non genera mana finché la condizione non è soddisfatta. Si attiva quando controlli 3 o più Avamposti in campo. Una volta attivo, genera 1 mana Est per turno. |
| **Utilizzo** | Illimitato — 1 volta per turno (solo se attivo) |
