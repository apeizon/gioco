# FORNACE DI KHAL

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Caleth |
| **Integrità** | 8 |
| **Effetto** | Quando entra in campo, infliggi 1 danno a un bersaglio a scelta. Genera 1 mana Sud. Può essere attaccato direttamente — le tue unità possono difenderlo. |
| **Utilizzo** | Illimitato — 1 volta per turno |
