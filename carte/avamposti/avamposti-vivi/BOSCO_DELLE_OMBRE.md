# BOSCO DELLE OMBRE

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Endal · Faelorn |
| **Integrità** | 8 |
| **Effetto** | Quando entra in campo, milla 1 carta dal tuo mazzo. Se in questo turno una creatura è morta, metti un segnalino +1/+1 su una creatura che controlli. Genera 1 mana Est o 1 mana Ovest a scelta. Può essere attaccato direttamente — le tue unità possono difenderlo. |
| **Utilizzo** | Illimitato — 1 volta per turno |
