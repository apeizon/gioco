# PIRA MALEDETTA

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Caleth · Endal |
| **Integrità** | 7 |
| **Effetto** | Quando entra in campo, infliggi 1 danno a un bersaglio e milla 1 carta dal tuo mazzo. Genera 1 mana Sud o 1 mana Est a scelta. Può essere attaccato direttamente — le tue unità possono difenderlo. |
| **Utilizzo** | Illimitato — 1 volta per turno |
