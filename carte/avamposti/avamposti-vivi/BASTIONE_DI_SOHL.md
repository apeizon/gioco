# BASTIONE DI SOHL

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Nord |
| **Integrità** | 10 |
| **Effetto** | Quando entra in campo, guarda l'ultima carta del tuo mazzo: scegli se tenerla lì o metterla in cima. Genera 1 mana Nord. Può essere attaccato direttamente dagli avversari — le tue unità possono difenderlo. |
| **Utilizzo** | Illimitato — 1 volta per turno |
