# Ombra Senza Nome

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Est |
| **Effetto** | Entra coperto. Non può essere counterato quando entra. Finché è coperto, non può essere bersaglio di nessun effetto. Quando lo riveli, genera 1 mana Est. |
| **Utilizzo** | Illimitato — 1 volta per turno |
