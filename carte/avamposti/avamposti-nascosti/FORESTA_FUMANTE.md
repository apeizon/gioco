# Foresta Fumante

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Sud / Ovest |
| **Effetto** | Entra coperto. Non può essere counterato quando entra. Finché è coperto, non può essere bersaglio di nessun effetto. Quando lo riveli, genera 1 mana Sud o 1 mana Ovest a scelta. |
| **Utilizzo** | Illimitato — 1 volta per turno |
