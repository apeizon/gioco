# Giardino Segreto

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Ovest / Centro |
| **Effetto** | Entra coperto. Non può essere counterato quando entra. Finché è coperto, non può essere bersaglio di nessun effetto. Quando lo riveli, genera 1 mana Ovest o 1 mana Centro a scelta. |
| **Utilizzo** | Illimitato — 1 volta per turno |
