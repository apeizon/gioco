# Gelo Silente

| Campo | Valore |
|---|---|
| **Tipo** | Avamposto |
| **Fazione** | Nord / Est |
| **Effetto** | Entra coperto. Non può essere counterato quando entra. Finché è coperto, non può essere bersaglio di nessun effetto. Quando lo riveli, genera 1 mana Nord o 1 mana Est a scelta. |
| **Utilizzo** | Illimitato — 1 volta per turno |
