# Raffica Gelida

| Campo | Valore |
|---|---|
| **Tipo** | Magia — Istante |
| **Fazione** | Nord |
| **Rarità** | Comune |
| **Costo** | 1 Nord più 1 mana qualsiasi |
| **Effetto** | Un'unità avversaria bersaglio viene congelata per 1 turno. *(Gelo: l'unità non può attaccare né attivare abilità per il numero di turni indicato; può ancora bloccare.)* |
