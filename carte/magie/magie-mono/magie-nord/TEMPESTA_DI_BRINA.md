# Tempesta di Brina

| Campo | Valore |
|---|---|
| **Tipo** | Magia — Sorcery |
| **Fazione** | Nord |
| **Rarità** | Non Comune |
| **Costo** | 3 Nord più 1 mana qualsiasi |
| **Effetto** | Tempesta di Brina congela ogni unità avversaria in campo per 1 turno. *(Gelo: l'unità non può attaccare né attivare abilità per il numero di turni indicato; può ancora bloccare.)* |
