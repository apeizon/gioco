# Inverno Eterno

| Campo | Valore |
|---|---|
| **Tipo** | Magia — Sorcery |
| **Fazione** | Nord |
| **Rarità** | Eterno |
| **Costo** | 4 Nord più 2 mana qualsiasi |
| **Effetto** | Inverno Eterno congela ogni unità avversaria in campo per 3 turni. Inoltre, ogni tua unità con Baluardo ottiene +1 atk/+1 def in modo permanente. *(Gelo: l'unità non può attaccare né attivare abilità per il numero di turni indicato; può ancora bloccare. Baluardo: tutte le tue altre unità ricevono +0 atk/+1 def mentre questa unità è in campo.)* |
