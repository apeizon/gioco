# Maledizione del Lich

| Campo | Valore |
|---|---|
| **Tipo** | Magia — Istante |
| **Fazione** | Est |
| **Rarità** | Rara |
| **Costo** | 2 Est più 1 mana qualsiasi |
| **Effetto** | Applica un contatore Maledizione -3 atk/-3 def a un'unità avversaria bersaglio. *(Maledizione: il contatore è permanente. Rimane attivo anche se l'unità lascia il campo o se la carta che lo ha applicato viene distrutta. Solo effetti che rimuovono contatori possono eliminarlo.)* |
