# Patto delle Radici

| Campo | Valore |
|---|---|
| **Tipo** | Magia — Istante |
| **Fazione** | Ovest |
| **Rarità** | Rara |
| **Costo** | 1 Ovest più 1 mana qualsiasi |
| **Effetto** | Una tua unità bersaglio riceve +0 atk/+3 def e guadagna Compromesso fino alla fine del turno. *(Compromesso: ogni danno che l'unità subisce viene assorbito come HP dal suo controllore.)* |
