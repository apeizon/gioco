# Doppia Fiamma

| Campo | Valore |
|---|---|
| **Tipo** | Magia — Istante |
| **Fazione** | Sud |
| **Rarità** | Non Comune |
| **Costo** | 1 Sud più 1 mana qualsiasi |
| **Effetto** | La prossima Magia Sud che giochi in questo turno viene copiata. Puoi scegliere nuovi bersagli legali per la copia. |
