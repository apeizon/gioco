# Altare del Vegliante Eterno

| Campo | Valore |
|---|---|
| **Tipo** | Santuario |
| **Fazione** | Nomade |
| **Rarità** | Non Comune |
| **Costo** | 3 mana qualsiasi |
| **Effetto** | All'inizio della propria fase di Mantenimento, ogni giocatore guarda la prima carta del proprio mazzo. Quel giocatore può rimettere quella carta in fondo al proprio mazzo. |
