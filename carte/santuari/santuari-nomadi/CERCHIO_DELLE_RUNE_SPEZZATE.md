# Cerchio delle Rune Spezzate

| Campo | Valore |
|---|---|
| **Tipo** | Santuario |
| **Fazione** | Nomade |
| **Rarità** | Rara |
| **Costo** | 4 mana qualsiasi |
| **Effetto** | Tutte le unità con **Scudo** (assorbe 1 fonte di danno o distruzione senza effetto) entrano in campo con un contatore Scudo aggiuntivo. |
