# Necropoli del Sussurro

| Campo | Valore |
|---|---|
| **Tipo** | Santuario |
| **Fazione** | Est |
| **Rarità** | Rara |
| **Costo** | 2 Est |
| **Effetto** | Una volta per turno, ogni giocatore può pagare 2 mana qualsiasi per restituire una creatura bersaglio dal proprio cimitero alla propria mano. |
