# Tomba del Falcemortis

| Campo | Valore |
|---|---|
| **Tipo** | Santuario |
| **Fazione** | Est |
| **Rarità** | Eterno |
| **Costo** | 1 Est + 3 |
| **Effetto** | Ogni volta che una carta entra nel cimitero di un giocatore, quel giocatore subisce 1 danno. All'inizio del Mantenimento di ogni giocatore, quel giocatore scarta la prima carta del proprio mazzo nel proprio cimitero. |
