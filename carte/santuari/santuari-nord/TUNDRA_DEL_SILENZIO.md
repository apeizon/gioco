# Tundra del Silenzio

| Campo | Valore |
|---|---|
| **Tipo** | Santuario |
| **Fazione** | Nord |
| **Rarità** | Non Comune |
| **Costo** | 1 Nord + 2 |
| **Effetto** | Nessuna creatura può attaccare nel turno in cui entra in campo. Le creature con **Velocità** (possono attaccare nello stesso turno in cui entrano in campo) perdono questa abilità mentre questo Santuario è in campo. |
