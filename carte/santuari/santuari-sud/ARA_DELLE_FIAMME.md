# Ara delle Fiamme

| Campo | Valore |
|---|---|
| **Tipo** | Santuario |
| **Fazione** | Sud |
| **Rarità** | Rara |
| **Costo** | 1 Sud + 3 |
| **Effetto** | Tutte le creature con costo totale pari o inferiore a 3 guadagnano **Frenesia** (questa unità può attaccare due volte per turno; la seconda volta non può essere bloccata). |
