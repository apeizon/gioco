# Ricorrenza del Deserto

| Campo | Valore |
|---|---|
| **Tipo** | Santuario |
| **Fazione** | Sud |
| **Rarità** | Eterno |
| **Costo** | 2 Sud + 2 |
| **Effetto** | All'inizio del Mantenimento di ogni giocatore, ogni creatura che quel giocatore controlla riceve un segnalino **-0/-1** permanente (riduce la DEF di 1 in modo cumulativo). Ogni creatura che riceve questo segnalino guadagna +2 ATK fino alla fine di quel turno. |
