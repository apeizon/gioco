# Tassa di Rowa

| Campo | Valore |
|---|---|
| **Tipo** | Tragedia |
| **Fazione** | Nomadi (incolore) |
| **Rarità** | Eterno |
| **Costo** | 4 |
| **Malus** | Ogni volta che peschi una carta, ogni avversario pesca 1 carta. |
| **Bonus** | Tutte le tue Magie costano 2 mana generici in meno (mai sotto 1 mana totale). |
| **Eco** | — (non ripetibile; effetti permanenti finché la Tragedia resta in campo). |
