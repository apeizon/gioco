# Patto del Viandante

| Campo | Valore |
|---|---|
| **Tipo** | Tragedia |
| **Fazione** | Nomadi (incolore) |
| **Rarità** | Comune |
| **Costo** | 2 |
| **Malus** | Subisci 5 danni. |
| **Bonus** | Peschi 2 carte. |
| **Eco** | 3 (paga 3 mana, una volta per turno, per ripetere l'effetto: subisci altri 5 danni e peschi altre 2 carte). |
