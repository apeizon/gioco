# Maledizione delle Rovine

| Campo | Valore |
|---|---|
| **Tipo** | Tragedia |
| **Fazione** | Nomadi (incolore) |
| **Rarità** | Rara |
| **Costo** | 3 |
| **Malus** | Tutte le tue creature subiscono -1 ATK fino alla rimozione della Tragedia. |
| **Bonus** | All'inizio del tuo Mantenimento, aggiungi 2 mana generici alla tua riserva mana (persiste fino a fine turno). |
| **Eco** | Metti una carta dalla tua mano in fondo al tuo mazzo (una volta per turno) per applicare una nuova istanza dell'effetto: -1 ATK cumulativo alle tue creature e +2 mana generici cumulativi a Mantenimento. |
