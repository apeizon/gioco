# Il Prezzo da Pagare

| Campo | Valore |
|---|---|
| **Tipo** | Tragedia |
| **Fazione** | Nord |
| **Rarità** | Rara |
| **Costo** | 2 Nord + 1 |
| **Malus** | Sacrifica una creatura che controlli a tua scelta. |
| **Bonus** | Peschi 3 carte. |
| **Eco** | Scarta una carta dalla tua mano (una volta per turno) per ripetere l'effetto: sacrifica un'altra creatura e peschi altre 3 carte. |
