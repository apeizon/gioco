# Tempesta di Sabbia

| Campo | Valore |
|---|---|
| **Tipo** | Tragedia |
| **Fazione** | Sud |
| **Rarità** | Eterno |
| **Costo** | 1 Sud + 3 |
| **Malus** | Alla fine di ogni tuo turno, subisci 3 danni. |
| **Bonus** | Tutte le tue creature guadagnano **Frenesia** (questa unità può attaccare due volte per turno; la seconda volta non può essere bloccata). |
| **Eco** | — (non ripetibile; effetti permanenti finché la Tragedia resta in campo). |
