# Forgia di Caleth

| Campo | Valore |
|---|---|
| **Tipo** | Artefatto |
| **Fazione** | Sud |
| **Rarità** | Rara |
| **Costo** | 2 Sud più 1 mana qualsiasi |
| **Effetto** | Tutte le tue creature hanno Surriscaldamento. *(Surriscaldamento: ogni volta che giochi una Magia Sorcery o Istante nel tuo turno, questa unità guadagna +1 atk/+0 def fino alla fine del turno.)* |
