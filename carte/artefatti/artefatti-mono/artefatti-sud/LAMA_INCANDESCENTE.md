# Lama Incandescente

| Campo | Valore |
|---|---|
| **Tipo** | Artefatto — Equipaggiamento |
| **Fazione** | Sud |
| **Rarità** | Non Comune |
| **Costo** | 1 Sud più 1 mana qualsiasi |
| **Equipaggia** | 2 mana qualsiasi |
| **Effetto** | La creatura equipaggiata guadagna Frenesia. *(Frenesia: l'unità può attaccare due volte in questo turno; il secondo attacco non può essere bloccato.)* |
