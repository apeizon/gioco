# Cinto di Endal

| Campo | Valore |
|---|---|
| **Tipo** | Artefatto — Equipaggiamento |
| **Fazione** | Est |
| **Rarità** | Non Comune |
| **Costo** | 2 Est più 1 mana qualsiasi |
| **Equipaggia** | 2 mana qualsiasi |
| **Effetto** | La creatura equipaggiata guadagna Revoca. *(Revoca: una volta per partita per la creatura equipaggiata, quando quella creatura va al Cimitero, il suo controllore può rimetterla in campo alla fine del turno con metà dei suoi ATK/DEF originali (arrotondati per difetto). Una stessa creatura non può ottenere il beneficio di Revoca più di una volta per partita, anche se equipaggiata di nuovo.)* |
