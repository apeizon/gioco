# Calice del Falcemortis

| Campo | Valore |
|---|---|
| **Tipo** | Artefatto |
| **Fazione** | Est |
| **Rarità** | Rara |
| **Costo** | 3 Est più 2 mana qualsiasi |
| **Effetto** | All'inizio del tuo turno, ogni giocatore avversario mette le prime 2 carte del proprio mazzo nel proprio Cimitero. **Tappa, paga 2 Est:** scegli una carta unità in un Cimitero qualsiasi con costo mana totale 3 o inferiore. Quella carta torna in campo sotto il tuo controllo, tappata. |
