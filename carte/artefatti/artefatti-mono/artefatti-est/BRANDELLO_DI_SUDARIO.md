# Brandello di Sudario

| Campo | Valore |
|---|---|
| **Tipo** | Artefatto |
| **Fazione** | Est |
| **Rarità** | Comune |
| **Costo** | 1 Est |
| **Effetto** | **Tappa, sacrifica Brandello di Sudario:** applica un contatore Maledizione -1 atk/-1 def a un'unità avversaria bersaglio. *(Maledizione: il contatore è permanente. Rimane attivo anche se l'unità lascia il campo o se la carta che lo ha applicato viene distrutta. Solo effetti che rimuovono contatori possono eliminarlo.)* |
