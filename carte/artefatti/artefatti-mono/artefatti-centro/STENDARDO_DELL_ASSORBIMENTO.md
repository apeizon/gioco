# Stendardo dell'Assorbimento

| Campo | Valore |
|---|---|
| **Tipo** | Artefatto |
| **Fazione** | Centro |
| **Rarità** | Comune |
| **Costo** | 1 Centro più 2 mana qualsiasi |
| **Effetto** | Ogni volta che un'unità avversaria viene distrutta (in qualsiasi modo: combattimento, effetti, sacrificio), una tua creatura bersaglio riceve +1 atk/+1 def in modo permanente. |
